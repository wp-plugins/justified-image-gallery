Pure HTML and CSS and Jquery Justified Image Gallery.


== Description ==

Flicker style Justified Image Gallery for your wordpress site. easy to use via shortcode anywhere and unlimited time to use same page.

Live Demo: http://kentothemes.com/demo/justified-image-gallery/justified-image-gallery/


<strong>Plugin Features</strong>

* Use via shortcode.
* Display any post type.
* Ajax load more post.
* Support any themes

 


== Installation ==

1. Install as regular WordPress plugin. <br />

use shortcode anywhere in your post or page as following.<br />

`[kento_justified_image_gallery kjg_id='mygrid'  posttype='post' ]`


<strong>kjg_id</strong> -> (Optional) To Diplay more than one gallery same page you need to use different id (use letter only and underscore. ex: mygrid)<br />
<strong>posttype</strong> -> (Optional) Which post type you want to display on gallery(Only One. ex: post or page or portfolio).<br />
<strong>bgcolor</strong> -> (Optional) Background Color For Grid Area.<br />
